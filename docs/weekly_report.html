📌 周度概览：一部两幕剧——从"七月开门红"到"AI雪崩"

本周A股上演了一场教科书级的多空反转。周一（6/30）和周二（7/1），市场沉浸在半年末流动性回暖和政策催化的乐观中，上证指数连续两日收红，最高触及4143点。但周三（7/2），一场由Meta"卖算力"引爆的全球性科技股抛售潮席卷而来——科创50单日暴跌7.70%，创业板指重挫5.71%，全市场主力资金净流出1218亿。周四（7/3）市场初步企稳，但分化加剧：大金融托住指数，中小盘题材活跃修复，而高位科技股仍在下探。

我的判断：这不是牛市的终结，而是一次"暴力风格切换"。上半年涨幅巨大的AI/半导体/算力赛道集中兑现获利盘，同时资金向贵金属、医药、电力等高防御板块迁徙。本周的核心矛盾是：流动性充裕（央行持续呵护）vs 科技股估值泡沫 vs 全球风险偏好骤降。

🔍 大盘走势判断

趋势定性：典型的"冲高—崩跌—弱修复"三阶段走势，而非简单的单边行情。

第一阶段（周一至周二）：流动性驱动的普涨。
半年末MPA考核结束，央行逆回购+MLF持续投放，跨月资金重回宽松窗口。八部门有色金属稳增长方案落地、全球半导体第二轮涨价潮为科技股提供催化。上证从4058→4112（+1.3%），科创50从2133→2208（+3.5%），成交量温和放大至3.7万亿。

第二阶段（周三）：事件驱动的恐慌式抛售。
周三的暴跌不是技术性调整，而是三重利空共振：① Meta宣布筹建云基础设施业务出售过剩AI算力（击碎"算力稀缺"叙事）→ 费城半导体指数前夜暴跌6%+ → 日韩触发熔断 → A股AI硬件链集体跌停；② 美国6月非农仅5.7万（预期11.3万），就业市场急刹车引发全球衰退担忧；③ 国内监管层密集发布融券做空新规+减持规范，高位股加大兑现压力。当日主力净流出1218亿，其中融资融券净流出1198亿（占98%），说明是杠杆资金踩踏式出逃，而非机构有序减仓。

第三阶段（周四）：大金融+贵金属稳住指数，科技继续失血。
上证从4020低点修复至4044（+0.37%），创业板基本持平，科创50再跌0.6%。市场呈现极致的杠铃策略：一端是银行/保险/贵金属等防御/避险品种，另一端是低位的中小盘题材股（汽车零部件、军工装备），中间的高位科技成长股仍在"失血"。

💡 核心判断：本轮调整的性质
我的判断是这是一次"暴力风格切换"而非"系统性熊市的开端"。理由有三：① 上证仅跌0.36%，沪深300大体持平，大市值品种获得支撑——真正的熊市不会出现如此极致的风格分化；② 周四市场已现企稳信号，涨停板108家，中小盘赚钱效应回暖——流动性并未系统性枯竭；③ 两融+北向周三集中出逃约2500亿后，周四恐慌情绪明显收敛。关键是：资金没有离场，只是在换地方。

🌍 与美股/港股的相关性：这次A股是被"带崩"的

本周A股和美股的分化格局非常微妙：道琼斯（价值）大涨1.11%创新高，纳斯达克（成长）跌1.45%——这恰好对应了A股上证（价值）仅跌0.36%而科创50（成长）暴跌7.37%的结构。A股和美股呈现的是"同风格共振、不同市场各自的风格内分化"。

我的判断：A股周三暴跌的主要驱动力是美股科技股的传导，而非独立的内生风险。Meta"卖算力"直接击碎了全球AI产业链的核心叙事——"算力稀缺"。这一叙事的崩塌是全球性的，A股的算力/光模块/半导体是这轮行情的核心品种，自然会首当其冲。但A股在周四出现了美股周三没有的"小票活跃修复"，说明A股内部的流动性韧性好于预期。

港股恒生指数本周涨1.22%，完全独立于A股的下跌——南向资金和外资对港股的低估值配置意愿仍然强劲。这是A股的一个安全垫：港股不崩，A股大金融/大盘价值就不会崩。

🌐 2.4 宏观/政策/事件分析

📰 本周最重要的3条新闻（附实质影响分析）

1️⃣ Meta宣布对外出售过剩AI算力（周二）

不是复述新闻：Meta这条消息改变了全球AI产业链的核心定价逻辑。"算力稀缺" → "算力过剩"的叙事切换是这轮暴跌的底层驱动。之前市场给AI算力链（光模块、服务器、交换机）的估值建立在"全球算力永远不够用"的假设上。Meta用行动告诉大家：巨头们在建的算力基础设施可能已经超出实际需求。这不是中国A股独立的风险——全球费城半导体指数一夜暴跌6%，日韩触发熔断，说明这是全球定价体系的重估。

对A股的实质影响：直接利好：AI应用端（模型使用成本下降）；直接利空：AI硬件链（光模块、算力租赁、服务器代工）。这个逻辑切换不是一天的交易能消化完的——未来1-2个月AI硬件链都将面临估值重构。

2️⃣ 美国6月非农仅5.7万，大幅低于预期（周三晚）

不是复述数据：非农从预期的11.3万断崖式下跌到5.7万，叠加4-5月数据同步下修，意味着美国就业市场可能正在从"过热"直接跳入"冷却"——跳过了"温和降温"。市场迅速定价：美联储年内不会加息，降息预期大幅升温。10年期美债收益率下行，美元指数跌破101。

对A股的实质影响：矛盾的双刃剑。利空面：就业急刹车 = 经济衰退担忧 → 全球风险偏好下行 → A股承压。利多面：降息预期+美元走弱 → 美元回流新兴市场 → 利好A股和人民币资产 → 利好黄金。本周A股走的是第一条逻辑（恐慌），但下周开始第二条逻辑（降息交易）可能逐渐占主导。关键看：A股能否"脱敏"于美股科技股的波动。

3️⃣ 证监会批复宇树科技科创板IPO + 收紧融券减持（周三）

不是复述政策：这一组合拳信号极为清晰——"引导资金从纯题材炒作转向硬科技价值投资"。宇树科技从受理到注册仅104天，创科创板最快纪录，说明监管层在加速优质硬科技企业上市，稀释存量题材股的稀缺性溢价。同时收紧融券做空和减持，压缩市场做空空间，遏制炒小炒差。这两条政策组合在一起，意味着A股的生态正在发生质变：钱要从"讲故事"流向"真业绩"。中报披露期恰逢其时——业绩兑现成为选股的唯一标准。
